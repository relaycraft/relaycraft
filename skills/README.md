# Skills Guide

Skills are reusable SOPs. Keep this folder small and practical.

## Principles

1. Load on demand only.
2. Keep only high-frequency, stable skills.
3. Delete stale/rare skills instead of adding folder complexity.

## Current core skills

- `react-component.md`
- `zustand-store.md`
- `tauri-command.md`
- `i18n-workflow.md`
- `error-handling.md`
- `commit-message-changelog.md`

## Add/Remove rule

- Add a skill only when a pattern repeats across multiple tasks.
- Remove a skill when usage is low or guidance is obsolete.
- Keep each skill short and execution-oriented (AI-first).
