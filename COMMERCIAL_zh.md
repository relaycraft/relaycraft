# 商业使用与支持

<p align="center">
  <a href="./COMMERCIAL.md">English</a> | <a href="./COMMERCIAL_zh.md">简体中文</a>
</p>

RelayCraft 自豪地采用 **AGPLv3** 协议开源。这确保了项目对社区保持免费和开放，同时也为项目的持续发展提供了一条可持续的道路。

## 许可模式

我们提供 **双重许可 (Dual Licensing)** 模式，以兼顾开源协作和专有商业需求。

| 特性 | 开源许可 (AGPLv3) | 商业许可 (Commercial) |
| :--- | :---: | :---: |
| **费用** | 免费 | 付费 |
| **源代码可用性** | **必须公开** (若修改并提供服务) | **私有 / 闭源** |
| **适用场景** | 个人, 开源项目, 非营利组织 | 企业, 专有产品集成 |
| **修改权** | 修改必须回馈社区 | 修改可以保持私有 |
| **技术支持** | 社区支持 (GitHub Issues) | 优先级邮件 / Slack 支持 |
| **协议兼容性** | 传播性强 (Copyleft), 不兼容闭源 | 兼容专有代码 |

### 1. 开源许可 (AGPLv3)
RelayCraft 对个人使用、教育用途以及从事开源项目的开发者完全免费。
- **要求**：如果您修改了 RelayCraft 并通过网络提供服务（SaaS），您**必须**在相同的 AGPLv3 许可下公开您的源代码。
- **适合人群**：独立开发者、学生、研究人员和开源贡献者。

### 2. 商业许可
如果您的组织希望在不受 AGPLv3 限制的情况下使用 RelayCraft（例如：将其集成到专有产品中、在不公开源码的情况下进行内部修改、或与闭源软件打包分发），我们提供 **商业许可**。

**权益包括：**
- **免除 AGPLv3 限制**：您可以保留您的集成代码和修改内容的私有权。
- **法律保障**：量身定制的许可协议，符合企业合规要求。
- **优先级支持**：直接对接核心开发团队。
- **功能定制**：优先排期对您业务至关重要的功能。

📧 **联系我们**：有关商业许可的咨询，请发送邮件至 **[license@relaycraft.dev](mailto:license@relaycraft.dev)**。
